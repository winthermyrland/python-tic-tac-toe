# Python Tic Tac Toe

## Project name
Python Tic Tac Toe

## Purpose
A simple GUI-based Tic Tac Toe game built with Python and tkinter.
Two human players take turns on a single machine.

## Repository
`winthermyrland/python-tic-tac-toe`

## Project-specific constraints
- Language: Python 3.13
- GUI: tkinter (stdlib only, no external GUI dependencies)
- Keep dependencies minimal — no pip packages required unless clearly justified
- Follow Python idiomatic conventions: snake_case for functions/variables, PascalCase for classes
